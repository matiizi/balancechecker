# Personal Finance Manager
Backend-приложение для управления личными финансами с авторизацией пользователей, учетом доходов/расходов и контролем бюджета.

## Основные возможности
- Авторизация пользователей - регистрация и вход по логину/паролю
- Учет финансов - добавление доходов и расходов с категориями
- Бюджетирование - установка лимитов бюджета по категориям расходов
- Статистика - детальная аналитика по доходам, расходам и бюджетам
- Умные оповещения - предупреждения о превышении бюджета и критическом балансе
- Сохранение данных - автоматическое сохранение в JSON файл при выходе

## Технологии
- Java - основной язык программирования
- Gson - сериализация и десериализация JSON данных
- Collections Framework - хранение данных в памяти
- File I/O - сохранение и загрузка данных из файлов

## Описание основных классов проекта
- Main.java - Точка входа, меню и основной цикл
- UserService.java - Сервис аутентификации и регистрации
- FileService.java - Работа с файлами (сохранение/загрузка)
- DataStorage.java - Модель для хранения всех данных
- User.java - Модель пользователя
- Wallet.java - Кошелек с транзакциями и бюджетами
- Transaction.java - Абстрактный класс транзакции
- Income.java - Модель дохода
- Expense.java - Модель расхода
- BudgetCategory.java - Модель категории бюджета

## Конфигурация
Файл данных
- Формат: JSON
- Имя файла: src/main/java/data/data.json
- Расположение: корневая директория приложения
- Автосохранение: после каждой значимой операции

## Запуск приложения
- Клонируйте репозиторий или откройте проект в вашей IDE
- Убедитесь в наличии Gson в classpath
- Запустите главный класс Main.java
- Приложение стартует в интерактивном режиме в консоли

## Использование сервиса
### Регистрация и вход
- При первом запуске выберите "Зарегистрироваться"
- Создайте логин и пароль
- Для последующих входов используйте "Войти"

### Управление финансами
- Добавление дохода - укажите сумму, категорию и описание
- Добавление расхода - аналогично доходу, с контролем бюджета
- Просмотр транзакций - полная история операций
- Настройка бюджетов - установите лимиты по категориям
- Статистика - аналитика расходов и остатков бюджета

### Система оповещений
- Критические - превышение лимита бюджета
- Позитивные - все бюджеты в норме
- Финансовое здоровье - проверка баланса (доходы vs расходы)

## Тестирование функциональности
### Регистрация и аутентификация:
- Создайте нового пользователя
- Войдите с правильными данными
- Попробуйте войти с неверным паролем

### Управление транзакциями:
- Добавьте несколько доходов с разными категориями
- Добавьте расходы в категории с установленным бюджетом
- Просмотрите историю всех операций

### Бюджетирование:
- Установите лимит бюджета для категории "Еда"
- Добавьте расходы в эту категорию
- Проверьте статистику - должны увидеть остаток бюджета

### Оповещения:
- Установите маленький лимит бюджета (например, 1000)
- Добавьте расход превышающий лимит
- Убедитесь, что получили предупреждение

### Сохранение данных:
- Добавьте несколько транзакций
- Выйдите из приложения
- Зайдите снова, данные должны сохраниться

## Особенности реализации
- In-memory хранение - данные хранятся в коллекциях Java во время работы
- JSON сериализация - Gson с кастомными адаптерами для сложных типов
- Абстрактные транзакции - единая система для доходов и расходов
- Валидация ввода - проверка корректности сумм и данных
- Автосохранение - данные сохраняются после каждой значимой операции
- Интерактивное меню - понятный текстовый интерфейс
